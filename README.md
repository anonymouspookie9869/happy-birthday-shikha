# 🧛‍♀️ Shikha's Vampire Birthday Surprise ✨

A cinematic, interactive, and romantic web application built specifically for **Shikha's Birthday**. This project is a digital gift that takes the user through a journey of memories, love, and a special proposal.

## 🌟 Features

### 1. **Cinematic Loading Experience**
- **Syncing Heartbeats:** A beautiful pre-loader that sets the mood with a heartbeat animation and progress bar.
- **Interactive Start:** Requires user interaction to "Open Your Heart," ensuring the background music plays perfectly.

### 2. **Atmospheric Background Music**
- Features a persistent, romantic piano soundtrack (`audio.mp3`) that plays throughout the entire experience.
- Includes a subtle, modern music player toggle in the top-right corner to control the audio.

### 3. **Interactive Birthday Cake**
- A virtual cake-cutting ceremony where Shikha can interact with the cake to proceed to the next surprise.

### 4. **Memory Gallery**
- A beautifully styled photo gallery designed to showcase special moments and shared memories.

### 5. **Heartfelt Message & Memory Jar**
- **Personalized Letter:** A deep, romantic message dedicated to "My Vampire, Shikha."
- **Interactive Memory Jar:** A special feature where she can reveal hidden memories one by one.

### 6. **The Final Proposal**
- A cinematic conclusion featuring a romantic proposal, complete with animations and a "Yes" interaction.

## 🛠️ Tech Stack

- **React 19**: For the core application logic.
- **Vite**: For lightning-fast development and optimized builds.
- **Tailwind CSS**: For modern, responsive styling with a dark, romantic theme.
- **Motion (Framer Motion)**: For smooth, cinematic transitions and floating heart animations.
- **Lucide React**: For elegant, consistent iconography.

## 🚀 How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-name>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to `http://localhost:3000`.

## 📦 Deployment

This project is optimized for deployment on **Vercel**. Simply connect your GitHub repository to Vercel, and it will automatically handle the build and deployment process.
